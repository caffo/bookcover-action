### Dependencies

- Node: `brew install node`
- ImageMagick: `brew install imagemagick`

### How to run

1. `npm install`
2. `npm start`

### Outcome

1. Available book covers saved to `/src/covers`
2. Book covers and tooltips added to `/src/Recently_ReadDatabase.html`
